# HawkParser

A recursive descent parser written in Java for a programming languages course.

## Overview
This project implements a parser in Java that reads input files, tokenizes them, and parses them according to the grammar rules defined in the program.

## Features
- Lexer for tokenizing input
- Parser for checking grammar structure
- Error handling with parse exceptions
- Multiple test input files included

## Files
- `HawkParser.java` - main source code
- `tests/input1.txt` to `tests/input6.txt` - sample test cases
- `README.md` - project documentation

## How to Compile
```bash
javac HawkParser.java
